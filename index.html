Thêm ứng dụng mới

Mở file index.html và thêm một dòng vào mảng APPS:

{ title: "Tên ứng dụng", file: "ten-file.html", emoji: "🎯", tag: "game", // game | edu | tool desc: "Mô tả ngắn về ứng dụng.", date: "2025", color: "#1a1a2a" // màu nền thumbnail }

Hướng dẫn host miễn phí

  • 1
    Tạo tài khoản tại github.com
  • 2
    Tạo repo mới, đặt tên ví dụ my-portal
  • 3
    Upload index.html và tất cả file .html vào cùng thư mục
  • 4
    Vào Settings → Pages, chọn branch main → Save
  • 5
    Truy cập ngay tại tenban.github.io/my-portal 🎉
