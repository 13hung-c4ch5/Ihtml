➕ Thêm file HTML của bạn

Để thêm app/game mới, mở file index.html và thêm một object vào mảng APPS ở đầu script:

{ title:"Tên app", file:"ten-file.html", emoji:"🎯", tag:"game", desc:"Mô tả ngắn", date:"2025" }

Sau đó đặt file HTML tương ứng cùng thư mục với index.html rồi upload lên GitHub Pages.
